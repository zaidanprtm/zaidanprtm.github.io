Penjelasan Elemen Tersusah dan Elemen yang Tidak Mungkin Diimplementasikan

Elemen Tersusah

1. Pembagian kolom dan row yang bagus dan sesuai. (Grid)


Awalnya saya kesusahan dalam membagi kolom dan row yang sesuai dengan mockup saya padahal di awal terkesan mudah. Hal ini cukup memakan waktu saya dalam mempelajari lecture yang diberikan Pak Eka dan juga browsing di Google.

2. Menempatkan Icon dan Kalimat Sejajar


Saya cukup banyak menggunakan icon dan icon tersebut diletakkan sejajar di samping kalimat. Pada awal implementasi saya menggunakan picture sebagai icon dan mengaturnya agar sejajar itu sulit sekali.

3. Memunculkan Popup Berhasil Setelah Registrasi


Hal ini cukup sulit karena pada awalnya ide saya malah membuat page baru hanya untuk memunculkan semacam popup berhasil.

Elemen yang Tidak Mungkin

Fixed Footer

Hal ini tidak bisa saya implementasikan karena ketika saya mencobanya error terjadi di beberapa page menyebabkan card menjadi sangat kecil dan tampilan rusak. Saya mencari-cari mengapa hal ini terjadi dan tidak menemukannya. Saya menggantinya dengan footer biasa, jadi ketika konten kurang atau resolusi layar besar sehingga konten naik maka footer akan sedikit naik dan tidak di bawah. Saya mengakalinya di beberapa page dengan tag br agar konten cukup panjang ke bawah.
